# goEngage
1. runs locally .
2. support for user input and stop exection of program .
3. can import play.golang.org links (ex:https://play.golang.org/p/uFLpwgy2Ug).
4. can export play.golang.org links .

3. ``PORT=9000 go run server.go ``
